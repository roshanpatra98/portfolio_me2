Text Format web application

TextFormat gives you a way to analyze your text quickly and efficiently. TextFormat is a free character counter tool that provides instant character count & word count statistics for a given text. TextFormat reports the number of words and characters. Thus it is suitable for writing text with word/ character limit.

This word counter software works in any web browsers such as Chrome, Firefox, Internet Explorer, Safari, Opera. It suits to count characters in facebook, blog, books, excel document, pdf document, essays, etc.
